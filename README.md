## 命令

#### 0. 安装

```bash
   yarn install
```

#### 1. 运行开发环境

```bash
   yarn start
```

#### 2. 输出生产环境文件

```bash
   yarn run build
```

#### 3. 输出生产环境，并查看模块分配

```bash
   yarn run build -- -r
```
