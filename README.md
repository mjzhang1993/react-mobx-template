# react-mobx-template

## 命令

#### 0. 安装

```bash
   sudo yarn install
```

#### 1. 运行开发环境

```bash
   yarn run dev # 或者 yarn start
```

#### 2. 输出生产环境文件

```bash
   yarn run build
```

#### 3. 输出生产环境，并查看模块分配

```bash
   yarn run build --report
```

#### 4. 运行本地开发服务器

```bash
   yarn run server
```

#### 5. 安装 yarn

> yarn 是用来代替 npm 的 node 包管理工具, 与 npm 有类似的命令，具体安装方式及使用方法查看 [yarn 命令使用](http://blog.csdn.net/mjzhang1993/article/details/70092902)
